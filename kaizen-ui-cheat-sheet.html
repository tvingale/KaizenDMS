Usage Rules (Quick)

  • Top bar is global; left sidebar is module-specific and collapsible.
  • Primary actions use brand red; only one primary per page.
  • Statuses use colored badges with contrasting text.
  • Filters are hidden behind a “Filter” button/panel.
  • Spinners for loading; toasts for light success; inline banners for critical changes.
  • Empty states: minimal by default; add guidance only when action is required.
